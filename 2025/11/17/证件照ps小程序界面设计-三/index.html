安装openCV和python识别脸、脖子、衣服(三)

发表于2025-11-17|更新于2025-11-21
|浏览量:

环境

QT6 + python3.14

安装命令

pip install --no-cache-dir -i https://pypi.tuna.tsinghua.edu.cn/simple numpy
pip install --no-cache-dir -i https://pypi.tuna.tsinghua.edu.cn/simple torch torchvision
pip install --no-cache-dir -i https://pypi.tuna.tsinghua.edu.cn/simple opencv-contrib-python==4.11.0.86

在https://opencv.org/releases 官网下载windows版本的可执行文件，文件名通常为opencv-4.x.x-vc14_vc15.exe，运行下载好的.exe文件，它实际上是一个自解压压缩包。你只需要指定一个目标文件夹，它将把预编译好的所有文件（包括库、头文件、示例等）解压到该目录。解压完成后，你就得到了预编译的OpenCV库。

环境配置

设置环境变量：为了方便，可以添加一个 OPENCV_DIR 系统环境变量，指向OpenCV的构建目录（例如 D:\OpenCV\Build\x64\vc16）。同时，将 %OPENCV_DIR%\bin 添加到系统的 Path 变量中，以便程序运行时能找到必要的DLL文件。

mingw编译器改为msvc:下载社区版msvc https://visualstudio.microsoft.com/zh-hans/free-developer-offers/ 安装时选择 “使用c++的桌面开发” “win10/11sdk” “cmake工具” “c++分析工具”
